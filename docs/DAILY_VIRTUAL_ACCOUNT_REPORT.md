# Daily Virtual Account Report

The Daily Virtual Account Report consolidates your local virtual trading activity.

It calculates:
- Starting and Ending Cash/Equity
- Realized and Unrealized PnL
- Gross and Net Exposures
- Order acceptance, fill, and rejection counts

## Commands
To produce the report from the CLI:
```bash
python -m usa_signal_bot paper-daily-report --latest-paper
```
